# connectors

Data-source connectors (scrapers, feeds, messaging), packaged as a
standalone Python library.

## Install

```bash
pip install -e .
```

For development (tests included):

```bash
pip install -e ".[dev]"
```

## Usage

```python
import connectors

print(connectors.__version__)
```

## Development

```bash
pytest
```
